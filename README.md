# Voces que Restauran

**Voces que Restauran** es un proyecto educativo de software creado como producto de transferencia y apropiación de los aprendizajes obtenidos durante la misión académica ICT Corea 2026.

Su primera fase consiste en convertir una unidad didáctica mediada por TIC en un recurso web interactivo, consultable por docentes, estudiantes y comunidad educativa.

La aplicación funcional se construirá posteriormente, a partir de los diseños, necesidades, prototipos y decisiones desarrolladas con estudiantes de 11 a 14 años del Liceo Caucasia.

## Propósito general

Usar el software como medio para sostener una experiencia pedagógica viva orientada a:

- visibilizar las historias de los estudiantes;
- fortalecer la convivencia escolar;
- promover prácticas pedagógicas restaurativas;
- desarrollar uso crítico, creativo y responsable de la inteligencia artificial;
- transferir el enfoque de Vibe Coding trabajado durante ICT Corea 2026.

## Fases

1. Unidad didáctica interactiva.
2. Implementación pedagógica.
3. Cocreación con estudiantes.
4. Construcción de la app.
5. Pilotaje y mejora.
6. Publicación y transferencia.
