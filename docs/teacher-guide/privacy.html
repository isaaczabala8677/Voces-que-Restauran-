Privacidad y protección

La experiencia se diseña para proteger a estudiantes y docentes. Se evita pedir información personal y se prioriza el uso de personajes y situaciones ficticias.

Para estudiantes

  • No compartas nombres reales ni relatos personales.
  • Trabaja con situaciones ficticias y generales.
  • Si tienes dudas, pide ayuda a la docente.

Para docentes

  • No se usan formularios ni almacenamiento de respuestas.
  • El recurso no reemplaza rutas institucionales ni procesos de convivencia.
  • Se recomienda acompañar el uso con lenguaje claro y orientaciones de cuidado.

Voces que Restauran

Recurso educativo en fase de prototipo. No recopila datos ni funciona como canal de denuncias o sistema de gestión de convivencia.
