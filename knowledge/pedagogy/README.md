# Biblioteca pedagógica

Esta carpeta contiene los elementos pedagógicos transversales de la unidad didáctica Voces que Restauran:

- estrategia de evaluación;
- evidencias de aprendizaje;
- rúbricas;
- productos de aprendizaje.

Estos documentos complementan la estructura formal almacenada en `knowledge/unit/` y deben ser consultados por Codex antes de diseñar las secciones de evaluación del recurso interactivo.
