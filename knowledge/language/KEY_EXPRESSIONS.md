# Expresiones clave

## Propósito

Este documento organiza expresiones breves en inglés y español para apoyar la comunicación durante la unidad **Voces que Restauran**.

Las expresiones funcionan como marcos de apoyo. Los estudiantes pueden:

- leerlas;
- completarlas;
- combinarlas;
- adaptarlas;
- utilizarlas en tarjetas;
- incorporarlas a diálogos;
- emplearlas durante presentaciones.

No se espera que todas sean memorizadas.
