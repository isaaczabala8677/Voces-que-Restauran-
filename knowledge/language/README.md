# Lenguaje, comunicación y recursos bilingües

## Propósito de la carpeta

Esta carpeta organiza los recursos lingüísticos en español e inglés que apoyan la implementación de la unidad didáctica **Voces que Restauran**.

Los recursos permiten que los estudiantes utilicen el lenguaje para:

- reconocer y expresar emociones;
- comunicar necesidades;
- escuchar diferentes perspectivas;
- formular preguntas;
- manifestar acuerdo o desacuerdo;
- analizar situaciones ficticias;
- proponer acciones de reparación;
- colaborar;
- describir prototipos;
- presentar ideas;
- construir mensajes de convivencia y cultura de paz.

El inglés no se incorpora como una lista aislada de contenidos gramaticales. Se utiliza como herramienta de comunicación dentro de actividades relacionadas con convivencia, participación, inteligencia artificial, diseño y cultura de paz.

## Principios del componente lingüístico

El lenguaje debe ser:

- comprensible;
- respetuoso;
- inclusivo;
- apropiado para la edad;
- conectado con situaciones comunicativas;
- apoyado mediante modelos;
- flexible;
- libre de expresiones humillantes;
- orientado a la escucha y la reparación.

---

## Nivel de referencia

Las expresiones se diseñan principalmente para estudiantes de nivel inicial o básico de inglés.

Se priorizan:

- palabras frecuentes;
- oraciones breves;
- estructuras repetibles;
- apoyos visuales;
- modelos;
- marcos de oración;
- trabajo colaborativo;
- comprensión e intención comunicativa.

No se espera que todos los estudiantes produzcan mensajes con la misma extensión o precisión.
