# Prompts educativos para Gemini

## Propósito

Este documento contiene prompts que pueden adaptarse para utilizar Gemini u otra herramienta de inteligencia artificial generativa durante la unidad **Voces que Restauran**.

Los prompts apoyan actividades pedagógicas relacionadas con:

- emociones;
- convivencia;
- enfoque restaurativo;
- inglés;
- cultura de paz;
- ciudadanía digital;
- diseño;
- evaluación crítica.

Todos los resultados deben revisarse antes de presentarse a los estudiantes.

---

## Instrucción de seguridad general

Esta instrucción puede añadirse al final de los prompts:

```text
Utiliza únicamente personajes y situaciones ficticias. No solicites ni inventes nombres completos, testimonios, datos personales, información familiar, diagnósticos o detalles que permitan identificar a estudiantes reales. No determines culpables, no recomiendes sanciones y no sustituyas la intervención docente o las rutas institucionales. Usa lenguaje respetuoso y apropiado para estudiantes de 11 a 14 años.
```

# Prompts para planeación docente

## Adaptar una actividad

```text
Actúa como diseñador pedagógico.

Contexto:
Estoy implementando una unidad interdisciplinaria llamada “Voces que Restauran” con estudiantes de secundaria entre 11 y 14 años. La unidad integra inglés, convivencia y tecnología.

Tarea:
Adapta la siguiente actividad para una sesión de [duración] minutos:

[DESCRIBIR ACTIVIDAD]

Incluye:
- propósito de aprendizaje;
- instrucciones paso a paso;
- materiales;
- apoyos para estudiantes con nivel inicial de inglés;
- alternativa desconectada;
- evidencia de aprendizaje;
- preguntas de reflexión;
- medidas de privacidad.

Restricciones:
No solicites relatos personales, nombres ni conflictos reales. No propongas diagnósticos, sanciones ni evaluación automatizada de emociones.
```

---

## Simplificar instrucciones

```text
Actúa como editor de materiales educativos.

Reescribe las siguientes instrucciones para estudiantes de 11 a 14 años:

[PEGAR INSTRUCCIONES]

Utiliza:
- frases breves;
- vocabulario sencillo;
- pasos numerados;
- un ejemplo;
- un recordatorio de privacidad.

Mantén el significado pedagógico y no elimines las medidas de seguridad.
```

---

## Crear una alternativa desconectada

```text
Actúa como diseñador de aprendizaje flexible.

Convierte la siguiente actividad digital en una alternativa completamente desconectada:

[DESCRIBIR ACTIVIDAD]

Debe poder realizarse con:
- papel;
- tarjetas;
- marcadores;
- trabajo por equipos.

Conserva:
- el propósito de aprendizaje;
- la participación;
- la reflexión;
- la protección de datos;
- la posibilidad de producir una evidencia.
```

# Prompts para casos ficticios

## Crear un caso de convivencia

```text
Actúa como creador de casos educativos.

Crea un caso completamente ficticio para estudiantes de 11 a 14 años sobre:

[TEMA: exclusión, rumor, desacuerdo de equipo, interrupciones, burla, uso inadecuado de un chat u otro tema]

El caso debe:
- tener entre 120 y 160 palabras;
- incluir tres personajes ficticios;
- permitir diferentes perspectivas;
- mostrar emociones y necesidades sin diagnosticarlas;
- evitar violencia grave;
- evitar nombres de instituciones reales;
- no presentar una solución final;
- terminar con una pregunta abierta.

Añade después:
1. hechos observables;
2. posibles interpretaciones;
3. posibles emociones;
4. posibles necesidades;
5. información que todavía no conocemos.

No determines quién tiene la culpa.
```

---

## Crear varias perspectivas

```text
A partir del siguiente caso ficticio:

[PEGAR CASO]

Escribe tres perspectivas breves, una por cada personaje.

Para cada perspectiva incluye:
- lo que pudo observar;
- lo que pudo pensar;
- cómo pudo sentirse;
- qué pudo necesitar;
- qué información desconoce.

Evita justificar acciones dañinas y evita presentar una perspectiva como la única correcta.
```

---

## Diferenciar hechos e interpretaciones

```text
Crea diez enunciados breves relacionados con un caso ficticio de convivencia escolar.

Cinco deben ser hechos observables y cinco deben ser interpretaciones.

Presenta:
1. el enunciado;
2. su clasificación;
3. una explicación breve.

Usa situaciones apropiadas para estudiantes de 11 a 14 años y no incluyas datos reales.
```

# Prompts sobre emociones y necesidades

## Crear tarjetas de emociones

```text
Crea doce tarjetas educativas bilingües sobre emociones.

Para cada tarjeta incluye:
- emoción en español;
- emoción en inglés;
- definición sencilla;
- ejemplo ficticio de una oración;
- posible necesidad relacionada;
- sugerencia de símbolo o imagen.

No presentes las emociones como diagnósticos ni asumas que una emoción siempre produce una conducta específica.
```

---

## Analizar emoción, pensamiento, necesidad y acción

```text
Crea cinco situaciones ficticias breves para trabajar esta secuencia:

situación → posible emoción → posible pensamiento → posible necesidad → opciones de acción

Para cada situación incluye:
- una respuesta que podría empeorarla;
- una respuesta respetuosa;
- una pregunta de reflexión.

Utiliza lenguaje apropiado para estudiantes de 11 a 14 años.
```

# Prompts de lenguaje restaurativo

## Crear preguntas restaurativas

```text
Actúa como asistente educativo con enfoque restaurativo.

Crea doce preguntas abiertas para analizar un caso ficticio de convivencia escolar.

Organízalas en estas categorías:
- qué ocurrió;
- pensamientos y emociones;
- personas afectadas;
- efectos;
- necesidades;
- responsabilidad;
- reparación;
- prevención.

Usa lenguaje sencillo, respetuoso y no acusatorio.

No incluyas preguntas para obtener confesiones, determinar culpables o recomendar sanciones.
```

---

## Revisar preguntas

```text
Revisa las siguientes preguntas:

[PEGAR PREGUNTAS]

Clasifica cada una como:
- restaurativa;
- cerrada;
- acusatoria;
- confusa;
- inapropiada.

Después:
- explica brevemente la clasificación;
- transforma las preguntas problemáticas en preguntas abiertas, respetuosas y apropiadas para estudiantes de 11 a 14 años.
```

---

## Comparar castigo y reparación

```text
Crea una tabla educativa para estudiantes de secundaria que compare:

- castigo;
- consecuencia;
- responsabilidad;
- reparación;
- prevención.

Para cada concepto incluye:
- definición sencilla;
- propósito;
- ejemplo ficticio;
- pregunta para reflexionar.

Aclara que el enfoque restaurativo no elimina las normas, las consecuencias ni la responsabilidad.
```

---

## Crear propuestas de reparación ficticias

```text
A partir de este caso ficticio:

[PEGAR CASO]

Propón cinco posibles acciones de reparación.

Para cada acción explica:
- qué efecto busca atender;
- por qué está relacionada con el daño;
- qué responsabilidad implica;
- qué limitación puede tener;
- cómo podría revisarse si ayudó.

No propongas humillaciones, confrontaciones forzadas ni acciones que sustituyan las rutas institucionales.
```

# Prompts para inglés

## Crear glosario bilingüe

```text
Actúa como docente de inglés para estudiantes de nivel A1–A2.

Crea un glosario bilingüe de veinte palabras sobre:

[TEMA]

Para cada palabra incluye:
- español;
- inglés;
- definición sencilla en español;
- oración breve en inglés;
- traducción de la oración;
- sugerencia de apoyo visual.

Utiliza vocabulario apropiado para estudiantes de 11 a 14 años.
```

---

## Crear frases A1–A2

```text
Crea quince expresiones breves en inglés nivel A1–A2 sobre:

[TEMA: escucha, respeto, emociones, reparación, trabajo en equipo, privacidad o retroalimentación]

Incluye:
- expresión en inglés;
- traducción natural al español;
- función comunicativa;
- ejemplo de situación en la que puede utilizarse.

Evita traducciones literales poco naturales.
```

---

## Crear un diálogo

```text
Crea un diálogo ficticio y breve entre tres estudiantes.

Tema:
[DESCRIBIR TEMA]

Requisitos:
- nivel de inglés A1–A2;
- máximo 18 intervenciones;
- expresiones de escucha;
- un desacuerdo respetuoso;
- una pregunta para comprender;
- una propuesta de colaboración;
- traducción al español después del diálogo.

No incluyas un conflicto grave ni información real.
```

---

## Revisar un mensaje bilingüe

```text
Revisa este mensaje bilingüe:

[PEGAR MENSAJE]

Analiza:
- claridad;
- nivel de inglés;
- naturalidad;
- correspondencia entre ambos idiomas;
- tono;
- adecuación para estudiantes de 11 a 14 años.

Después propone una versión mejorada y explica únicamente los cambios principales.
```

# Prompts para cultura de paz

## Crear mensajes para una campaña

```text
Crea veinte mensajes bilingües breves para una campaña escolar sobre:

[TEMA]

Los mensajes deben:
- tener máximo doce palabras por idioma;
- promover una acción concreta;
- utilizar inglés nivel A1–A2;
- evitar culpabilizar;
- ser apropiados para afiches o tarjetas;
- incluir español e inglés.

Agrúpalos en cuatro categorías.
```

---

## Revisar mensajes de paz

```text
Revisa los siguientes mensajes:

[PEGAR MENSAJES]

Para cada mensaje analiza:
- claridad;
- tono;
- intención;
- posibilidad de culpabilización;
- naturalidad del inglés;
- relación con una acción concreta.

Marca:
- conservar;
- ajustar;
- descartar.

Propón una versión mejorada cuando sea necesario.
```

# Prompts para alfabetización en IA

## Explicar IA generativa

```text
Explica qué es la inteligencia artificial generativa para estudiantes de 11 a 14 años.

Incluye:
- una definición sencilla;
- una analogía;
- tres usos posibles;
- tres limitaciones;
- tres riesgos;
- cinco reglas de cuidado;
- una actividad de comprobación.

Evita decir que la IA piensa, siente o comprende como una persona.
```

---

## Crear una respuesta deliberadamente imperfecta

```text
Crea una respuesta ficticia de una inteligencia artificial sobre un desacuerdo escolar.

La respuesta debe contener, de manera moderada y apropiada para una actividad:
- una suposición no sustentada;
- una generalización;
- una recomendación demasiado rápida;
- ausencia de una perspectiva;
- una frase que requiera reformulación.

Después crea una guía docente que identifique esos problemas.

No incluyas contenido dañino ni un caso real.
```

Este prompt debe utilizarse únicamente bajo revisión docente.

---

## Comparar dos respuestas

```text
Genera dos respuestas diferentes ante este caso ficticio:

[PEGAR CASO]

Respuesta A:
Debe ser superficial y centrarse demasiado en el castigo.

Respuesta B:
Debe utilizar preguntas abiertas, reconocer perspectivas y orientar hacia responsabilidad y reparación.

Después crea cinco preguntas para que los estudiantes comparen ambas respuestas.

Mantén un lenguaje apropiado y evita recomendaciones para casos graves.
```

# Prompts para diseño

## Generar alternativas de solución

```text
Actúa como facilitador de diseño educativo.

Necesidad:

[DESCRIBIR NECESIDAD GENERAL]

Propón ocho alternativas de recursos pedagógicos o tecnológicos.

Para cada alternativa incluye:
- propósito;
- usuario;
- función principal;
- ventaja;
- riesgo;
- posibilidad de funcionar sin recopilar datos;
- alternativa desconectada.

No propongas vigilancia, clasificación de estudiantes, análisis emocional, perfiles ni sanciones automáticas.
```

---

## Revisar una idea de aplicación

```text
Analiza esta idea de prototipo:

[DESCRIBIR IDEA]

Evalúa:
- relación con la necesidad;
- utilidad pedagógica;
- claridad;
- privacidad;
- accesibilidad;
- riesgos;
- viabilidad para un prototipo estudiantil;
- funciones innecesarias;
- alternativa desconectada.

Organiza la respuesta en:
- fortalezas;
- aspectos por aclarar;
- riesgos;
- funciones que deben eliminarse;
- recomendaciones.
```

---

## Crear historias de usuario

```text
A partir de esta necesidad:

[DESCRIBIR NECESIDAD]

Crea cinco historias de usuario con la estructura:

Como [tipo de usuario],
necesito [acción o necesidad],
para [beneficio].

Los usuarios deben describirse de manera general. No incluyas nombres, diagnósticos ni perfiles sensibles.

Después identifica cuál historia es más adecuada para un primer prototipo y explica por qué.
```

# Prompts para evaluación y retroalimentación

## Crear una lista de cotejo

```text
Crea una lista de cotejo para revisar:

[PRODUCTO O ACTIVIDAD]

Debe incluir entre ocho y doce criterios relacionados con:
- propósito;
- claridad;
- enfoque restaurativo;
- uso del inglés cuando corresponda;
- privacidad;
- accesibilidad;
- colaboración;
- posibilidad de mejora.

Utiliza preguntas que puedan responderse con:
- sí;
- parcialmente;
- todavía no.

No incluyas puntuaciones de conducta.
```

---

## Generar retroalimentación formativa

```text
Actúa como docente que ofrece retroalimentación formativa.

Producto:

[DESCRIPCIÓN ANÓNIMA DEL PRODUCTO]

Criterios:

[PEGAR CRITERIOS]

Genera retroalimentación con esta estructura:
- dos aspectos que ya funcionan;
- evidencia que los demuestra;
- un aspecto prioritario por mejorar;
- una acción concreta;
- una pregunta para el equipo.

No asignes una calificación y no hagas afirmaciones sobre la personalidad de los estudiantes.
```

# Lista para revisar respuestas de Gemini

Antes de utilizar una respuesta, revisar:

- ¿Responde al prompt?
- ¿Es correcta?
- ¿Está completa?
- ¿Es apropiada para la edad?
- ¿Utiliza lenguaje respetuoso?
- ¿Presenta estereotipos?
- ¿Hace suposiciones?
- ¿Incluye información inventada?
- ¿Solicita datos?
- ¿Sugiere sanciones?
- ¿Confunde reparación con perdón obligatorio?
- ¿Necesita adaptación lingüística?
- ¿Qué debe decidir el docente?
- ¿Qué parte debe descartarse?

```text
Actividad:

Propósito:

Herramienta:

Respuesta obtenida:

Errores identificados:

Cambios realizados:

Decisiones humanas:

Aprendizaje:
```

No deben conservarse respuestas que contengan información personal.

## Criterio de actualización

Este documento debe actualizarse cuando:

- se incorporen nuevas actividades;
- cambie la herramienta;
- se identifiquen nuevos riesgos;
- se mejoren los prompts;
