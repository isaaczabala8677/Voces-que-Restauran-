# Prompts para inteligencia artificial y Vibe Coding

## Propósito de la carpeta

Esta carpeta organiza los prompts que pueden utilizarse durante la unidad didáctica **Voces que Restauran**.

Los prompts están diseñados para apoyar:

- generación de ideas;
- creación de casos ficticios;
- elaboración de mensajes bilingües;
- formulación de preguntas restaurativas;
- revisión de textos;
- diseño de prototipos;
- análisis de interfaces;
- generación inicial de código;
- prueba y mejora de soluciones.

Los prompts son puntos de partida. Deben ser revisados, adaptados y supervisados por el docente antes de utilizarse con estudiantes.

## Qué es un prompt

Un prompt es una instrucción que comunica a una herramienta de inteligencia artificial:

- el contexto;
- el propósito;
- el público;
- la tarea;
- el formato esperado;
- las condiciones;
- las restricciones;
- los criterios de calidad.

Un prompt claro no garantiza una respuesta correcta.

Todo resultado debe revisarse.
